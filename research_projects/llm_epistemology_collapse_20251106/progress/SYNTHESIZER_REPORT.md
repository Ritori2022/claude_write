## 📚 内容结构

### 第一部分：炼金术的五大核心特征

#### 1️⃣ **秘传知识（Tacit Knowledge）**
- **炼金术特征**：师徒秘传、隐喻指示、故意保密
- **对应关系**：
  - Reddit/Twitter的"咒语库"vs 中世纪的秘方
  - 80%的有效prompt来自"试错和社区分享"
  - 没有统一的prompt工程师培养课程
  - 最有效的system prompt通过"口碑传播"而非官方文档

#### 2️⃣ **核心迷思：转化 vs 涌现**
- **炼金术**：铅→金的本质转化
- **LLM**：Emergence（涌现）——21世纪的贤者之石
  - Wei等人的《涌现能力的三棱镜》（2023）
  - 2024年Schaeffer的批判：涌现可能是测量伪影
  - 核心问题：**我们观察现象但缺乏因果机制**

#### 3️⃣ **仪式化的实践**
- **炼金术**：特定时间表、符号、咒语、步骤执着
- **Prompt工程**：
  - 格式强迫症（System/Context/Task/Examples的特定顺序）
  - "让我们逐步思考"的魔法短语
  - temperature=0.7, top_p=0.9的"标准"参数
  - 通过试错发现→社区传播→形成"最佳实践"

#### 4️⃣ **不可重复性困境**
- **炼金术**：环境变异、仪器误差、记录不精确
- **LLM**：
  - Temperature/随机性导致同样prompt的不同输出
  - Anthropic内部研究（2024）：相同prompt在10次运行中差异35-45%
  - 模型版本升级导致历史prompt失效
  - 论文中只有12%提供足够细节来重现实验

#### 5️⃣ **缺失的理论基础**
- **炼金术失败**：不知道元素、化学键、反应机制
- **LLM危机**：
  - 黑盒性质（数十亿参数的矩阵乘法）
  - Mechanistic Interpretability仅能解释<124M参数的模型
  - 无法进行因果干预预测（只能运行→观察→事后分析）

### 第二部分：化学革命的三个转折点（及其缺席）

#### 🧪 **转折1：波义耳的实验规范化（1661）**
- **历史事实**：《怀疑的化学家》
- **关键原则**：
  - 实验可重复性必须是关键标准
  - 定量测量（称重）比定性描述更重要
  - 详细记录每个变量和条件
  - 公开发表而非保密

- **LLM的缺失**：
  - ✗ 详细的实验议定书
  - ✗ 对随机性的系统控制
  - ✗ 完整的失败报告（幸存者偏差严重）
  - ✗ 可重复的prompt（版本变化、API更新）

#### ⚖️ **转折2：拉瓦锡的定量化革命（1770s）**
- **历史成就**：质量守恒定律、化学方程式、元素概念
- **核心意义**：从"这发生在我身上"到"如果你按这些步骤做，这将发生"

- **LLM的缺失**：
  - 没有"模型方程"能预测特定prompt的结果
  - Chinchilla缩放定律仅预测能力规模，不能预测具体能力
  - 无法帮助设计更好的prompt
  - Prompt工程充满**特定案例**而缺乏**可泛化的原则**

#### 🔍 **转折3：可伪证性与范式确立（19-20世纪）**
- **哲学转变**：接受波普尔的可伪证性原则
- **关键信念**：理论可以是错的，实验可以驳斥理论

- **LLM的问题**：
  - "涌现"是**不可伪证的**假说
  - 无论观察如何，理论总是"正确的"
  - 对批评的耐心不足（Schaeffer vs Wei的争论）
  - "这需要特殊训练"的事后调整

### 第三部分：为什么Prompt工程还停留在炼金术阶段

#### 根本障碍1：理论的根本性缺失
- 炼金术失败于缺乏原子论和化学键的理论
- LLM无法进步直到理解：
  - "理解"在神经网络术语中的本质
  - 为什么token序列产生不同结果
  - 新能力何时涌现

#### 根本障碍2：经济激励与秘密
- 化学成功因为科学提供了更好的结果
- AI领域的激励相反：
  - 公司对prompt保密（竞争优势）
  - 再现了炼金术的**知识囤积模式**
  - 最好的prompt通过社区口碑而非学术出版

#### 根本障碍3：模型的持续变化
- 化学受益于：化学规律永恒（1700年的反应今天仍有效）
- LLM面临：
  - GPT-3有效的prompt可能对GPT-4无效
  - 模型更新使知识库成为历史遗物
  - 无法建立统一理论

#### 根本障碍4：规模的不可控性
- 化学：简单的桌面实验，任何人都能做
- LLM：
  - 实验需要大量计算资源
  - 只有富有的组织能进行认真实验
  - 知识集中在少数几家公司
  - 独立科学验证几乎不可能

### 第四部分：三个原创思想实验

#### 💭 **思想实验1：波义耳访问2025**
**场景**：波义耳时光旅行，观察prompt工程师工作
**他的批评**：
1. 不可重复——无法控制变异
2. 不可测量——凭直觉判断"更好"
3. 缺乏理论——不知道"屡获殊荣"为何重要
4. 秘密化——成功无法教给他人

**波义耳的处方**：
- 标准化一切——精确定义"好故事"，用数字衡量
- 重复实验——每个prompt至少运行10次
- 发表失败——特别是不起作用的尝试
- 构建理论——基于观察提出可检验的假说

#### 💭 **思想实验2：Prompt质量守恒定律**
**假说**：固定模型的总性能指标是恒定的
```
性能总和(准确性、简洁性、诚实性、速度) = 常数
```

**可检验的预测**：
```
如果 prompt A 表现为：
准确性: 95%, 长度: 200词, 延迟: 1.2秒

那么，任何修改都遵循：
准确性' × 长度' × 速度' = 常数
```

**意义**：将prompt工程从"无限尝试"转变为"约束优化"——这是工程，而非巫术

#### 💭 **思想实验3：Prompt元素周期表**
**假想**：prompt可被分解为基础"元素"

**系统元素周期表**：
- 角色扮演 (Rp) - "你是一个X"
- 指导方法 (Im) - "你使用方法Y"
- 约束条件 (Co) - "避免Z"
- 输出格式 (Fo) - "以JSON返回"
- 上下文 (Cx) - "背景信息"
- 示例 (Ex) - "如果输入A，输出B"
- 温度调整 (Te) - "创意水平"

**化学式示例**：
- 数学prompt：Rp(Expert) + Im(Step-by-step) + Ex(2) → 高准确性
- 创意prompt：Rp(Writer) + Te(0.8) + Co(避免俗套) → 高创意性

**可预测性**：教学生"prompt公式"而非"prompt技巧"
